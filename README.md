## 🚀 Deploy ke Vercel (3 Langkah)

### Langkah 1 — Push ke GitHub
```bash
git init
git add .
git commit -m "feat: portfolio digdaya x hackathon"
git branch -M main
git remote add origin https://github.com/USERNAME/portfolio-digdaya.git
git push -u origin main
```

### Langkah 2 — Import di Vercel
1. Buka [vercel.com](https://vercel.com)
2. Klik **"Add New → Project"**
3. Import dari GitHub → pilih repo `portfolio-digdaya`
4. Klik **"Deploy"** — selesai! ✅

### Langkah 3 — Domain Custom (Opsional)
Di Vercel dashboard → Settings → Domains → tambahkan domain kamu.
